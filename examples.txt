<h3>Overview</h3>
There are 3 ways to issue commands to stellard.
<ul>
  	<li><strong>Command line</strong>- This is the usual method when you are running stellard locally.</li>

 	<li><strong>RPC</strong>- stellard can be configured to listen for http requests on an arbitrary port. You can then make normal PUT requests with the command and parameters sent as a JSON string in the data.</li>

  	<li><strong>Websocket</strong>- stellard can also be configured to create a websocket. You can send all the RPC commands through the websocket. It is useful if you want to subscribe to events from stellard like when a particular account gets a transaction.</li>

</ul>
